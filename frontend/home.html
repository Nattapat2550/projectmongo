🌟 My Web App

เว็บแอปพลิเคชันที่สร้างขึ้นด้วยเทคโนโลยีสมัยใหม่ เพื่อให้ประสบการณ์การใช้งานที่ดีที่สุด

✨ คุณสมบัติเด่น

  • 🔐 ระบบสมาชิกที่ปลอดภัยด้วย JWT Authentication
  • 🔗 เข้าสู่ระบบด้วย Google OAuth
  • 👤 จัดการข้อมูลส่วนตัวได้อย่างง่ายดาย
  • 🎨 เปลี่ยนธีมระหว่างโหมดสว่างและมืดได้
  • 📷 อัปโหลดและเปลี่ยนรูปโปรไฟล์ได้
  • 📱 Responsive Design รองรับทุกอุปกรณ์
  • 🚀 Performance ที่เร็วและเสถียร

🛠️ เทคโนโลยีที่ใช้

Frontend

HTML5, CSS3, JavaScript (ES6+)

Backend

Node.js, Express.js

Database

MongoDB with Mongoose ODM

Authentication

JWT, Passport.js, Google OAuth

🔒 ความปลอดภัย

เรามุ่งมั่นในการปกป้องข้อมูลของคุณด้วย:

  • การเข้ารหัสรหัสผ่านด้วย bcrypt
  • JWT Token สำหรับการตรวจสอบสิทธิ์
  • CORS Policy ที่เข้มงวด
  • Input validation และ sanitization
  • Secure file upload
